## 프로젝트 소개


<b>학수고대</b>는 SNS의 한계와 부작용을 인식하며 시작된 프로젝트입니다.

현대 사회에서 SNS는 주로 소통과 연결을 제공하는 도구로 간주됩니다. 그러나 그 안에는 목적 없는 자기 과시. 혹은 비교에 따른 우울감과 같은 문제가 발생하고 있습니다.

이러한 현상을 안타깝게 느끼며, 서로에게 격려와 지지를 제공할 수 있는 소통의 공간을 만들고자 서비스를 기획하게 되었습니다.

저희의 온라인 롤링페이퍼 서비스 [학수고대]는 개인이 진정으로 이루고 싶은 것을 깨닫고, 서로의 성장을 응원하는 과정을 통해 협력과 소통을 촉진하고자 합니다.

## 주요 기능


### [가입 및 로그인]
- 카카오 로그인을 활용했습니다.
    - 처음 가입하는 유저는 닉네임 설정 페이지로 이동합니다.
    - 이미 가입을 한 유저는 메인 페이지로 이동합니다.

### [유리병 생성]
- 응원을 받을 유리병을 생성할 수 있습니다.

    - 디데이를 설정합니다.
    - 유리병 디자인 및 종이학 개수 공개 여부를 선택합니다.

### [유리병 메인 페이지]
- 유리병의 주인과, 주인이 아닌 사람이 접근했을 때 다른 화면이 렌더링 됩니다.
    - 유리병의 주인이 접근하면, 편지를 확인할 수 있고, 공유하기 버튼이 렌더링 됩니다.
    - 주인이 아닌 사람이 접근하면, 종이학 보내기 버튼과 본인 유리병 확인 버튼이 렌더링 됩니다.

### [종이학 확인 및 유리병 공유]
-   유리병의 주인만 할 수 있습니다.
    - 아직 확인하지 않은 종이학은 노란색 아이콘을 통해 알 수 있습니다.
    - 공유하기를 클릭하면 해당 유리병 링크가 복사됩니다.

### [로딩 및 에러 페이지]
- 요청 및 렌더링을 기다릴 때 로딩 페이지가 나타납니다.
- 에러 페이지는 4가지 타입을 가집니다.
    - 로그인 미확인: 로그인을 하지 않은 유저가 메인 페이지 등에 접근할 때 나타납니다.
    - 오류: 서버에 요청을 실패했을 때 나타납니다.
    - 삭제: 삭제된 유리병에 접근했을 때 나타납니다.
    - 잘못된 경로: 정의되지 않은 경로로 접근할 때 나타납니다.
